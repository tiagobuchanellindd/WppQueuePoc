# WPP POC Documentation

This directory contains documentation for execution, architecture, planning, and study notes.

## Structure

- `docs/guides/`: conceptual and technical guides.
  - `docs/guides/01-objectives-summary.md`
  - `docs/guides/02-business-overview.md`
  - `docs/guides/03-architecture-styles.md`
  - `docs/guides/04-native-methods-guide.md`

- `docs/project/`: planning and status tracking.
  - `docs/project/01-wpp-poc-plan.md`
  - `docs/project/02-phase-status.md`

- `docs/operations/`: runbooks and execution steps.
  - `docs/operations/01-poc-operation-guide.md`

- `docs/estudo/`: Portuguese study notes by source file (already organized).

## Recommended reading order
1. `docs/guides/01-objectives-summary.md` (what this POC delivers)
2. `docs/guides/02-business-overview.md` (why this matters and business behavior)
3. `docs/guides/03-architecture-styles.md` (how the solution is organized and executed)
4. `docs/guides/04-native-methods-guide.md` (how Winspool interop works in this codebase)
5. `docs/project/01-wpp-poc-plan.md` (implementation strategy and scope)
6. `docs/project/02-phase-status.md` (current completion status by phase)
7. `docs/operations/01-poc-operation-guide.md` (how to execute commands and validated scenarios)
